# Snaptuki Domain Analysis

This folder documents the domain analysis for Snaptuki.

Snaptuki is a care coordination and safety platform for nursing homes. The goal of this domain analysis is to understand the business problem,
identify subdomains, define the core domain, model bounded contexts, and create a shared language between business and technical stakeholders.

## Main Domain

Nursing Home Care Operations and Safety Coordination

## Main Problem

Nursing homes need a reliable way to coordinate, monitor, document, and improve daily care work so that avoidable mistakes, missed routines, 
poor handovers, and delayed incident responses are reduced.
